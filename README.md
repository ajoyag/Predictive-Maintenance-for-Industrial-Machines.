# Predictive-Maintenance-for-Industrial-Machines.

Built a simple ML-based system that predicts whether a machine is healthy or at risk of failure using industrial sensor data.

Implemented and compared:

* Logistic Regression
* Decision Tree
* Naive Bayes

The project also included:

* Data preprocessing
* Feature scaling
* K-Fold Cross Validation
* Confusion Matrix
* Accuracy, Precision, Recall, and F1 Score evaluation
* Prediction graphs and model comparison

This project helped me understand how Machine Learning can be applied in Mechatronics and industrial automation systems for predictive maintenance.
